# 💰 Personal Finance Manager

A command-line Personal Finance Manager built with Python to help users manage income and expenses.

A modular command-line Personal Finance Manager built in Python to manage income and expenses with persistent storage, search, reporting, and data export capabilities.

---

## Features

- Add Income
- Add Expense
- View All Transactions
- Search Transactions
- Edit Transactions
- Delete Transactions
- Filter Transactions
- Monthly Summary
- Save Data using JSON
- Load Data from JSON
- Export Transactions to CSV
- Input Validation
- Modular Project Structure

---

## Technologies Used

- Python 3
- JSON
- CSV
- File Handling
- Modular Programming

## How to Run

Clone the repository:

```bash
git clone <repository-link>
```

Move into the project directory:

```bash
cd Personal-Finance-Manager
```

Run:

```bash
python main.py
```
